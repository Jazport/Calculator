# Cherry Blossom Calculator

A fully responsive, animated, cherry-blossom–themed calculator built using **HTML, CSS, and JavaScript**.  
Includes a history log, keyboard support, custom flower-button animations, and accessibility features.

---

## Features

### Calculator Functions
- Add, subtract, multiply, divide  
- Percent button works **inside expressions** (e.g., `80 - 40%`)  
- Delete (⌫) and Clear (C)  
- Decimal support  
- Error handling (invalid input, division by zero)

---

## UI / Styling
- Custom **flower-shaped animated buttons**  
- Pastel pink aesthetic  
- Smooth hover & press animations  
- Fully responsive mobile + desktop layout  
- Calculator stays centered, with history floating beside it  

---

## History Log
- Opens beside the calculator  
- Shows previous expressions + results  
- Click any history item to restore it  
- Clear-history button  
- Smooth slide-in animation  

---

## Keyboard Support
- Numbers: `0–9`  
- Operators: `+`, `-`, `*`, `/`  
- **Enter** → equals  
- **Backspace** → delete  
- **Escape** → clear  
- **Percent (%)** supported  

---

## Accessibility
- Strong visible **focus outlines** for keyboard navigation  
- High contrast content  
- Fully keyboard-operable  

---

## Project Files
- `index.html`  
- `style.css`  
- `script.js`  
- `README.md`  

---

## How to Run the Project
1. Download or clone the project.  
2. Open **index.html** in any browser.  
3. Use the flower buttons or your keyboard.  
4. Press **HX** to open the history log.
